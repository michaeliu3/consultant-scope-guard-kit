# Consultant Scope Guard Kit

Independent consultants can lose margin when small client requests expand a
fixed-scope project without a shared written record. This repository hosts a
free, fictional [scope change request example][example] that shows one compact
way to record the requested outcome, schedule and fee impact, dependencies, and
approval decision.

The paid **Consultant Scope Guard Kit** contains seven editable Markdown and CSV
templates for project scope, change requests, decisions, weekly client updates,
and handoff. It is a one-time **$19** download with no subscription or
proprietary file format.

- [Read the free filled-in example][example]
- [View the complete editable kit][product]

These are general project-operations aids, not legal advice or a substitute for
a contract reviewed for your situation. They do not guarantee payment, prevent
disputes, or produce a particular business outcome.
