# Consultant Scope Guard Kit

Use these files as prompts for a clear client conversation. Copy them for each
project, replace bracketed text, and keep dated approvals with the project
record. They are operational templates, not legal advice.

Suggested order:

1. Complete `scope-worksheet.md` before work begins.
2. Record material choices in `decision-log.csv`.
3. Send `weekly-client-update.md` on a consistent day.
4. Use `change-request.md` before accepting out-of-scope work.
5. Close with `handoff-checklist.md`.

See `worked-example.md` for a fictional end-to-end example showing how one
change moves through the worksheet, decision log, weekly update, and handoff.
